Wichtige HTML-Befehle (Tags) - Erklärung und Beispiele

Diese Seite bietet eine Übersicht über grundlegende HTML-Tags, ihre Funktionen und Beispiele, um Anfängern den Einstieg in HTML zu erleichtern.

Important HTML Commands (Tags) - Explanation and Examples

This page provides an overview of basic HTML tags, their functions, and examples to help beginners get started with HTML.



1. Grundstruktur eines HTML-Dokuments

Jedes HTML-Dokument hat eine grundlegende Struktur, damit der Browser weiß, wie er den Inhalt interpretieren soll.

<!DOCTYPE html>
<html lang="de">
    <head>
        <title>Seitenüberschrift</title>
    </head>
    <body>
        Inhalt der Webseite.
    </body>
</html>
  • <!DOCTYPE html>: Kennzeichnet ein HTML5-Dokument.
  • <html>: Umschließt den gesamten HTML-Code.
  • <head>: Beinhaltet Metadaten, Skripte, Stylesheets und den Titel.
  • <title>: Titel der Webseite, im Browser-Tab sichtbar.
  • <body>: Enthält den eigentlichen sichtbaren Inhalt.

Inhalt der Webseite.

1. Basic Structure of an HTML Document

Every HTML document has a basic structure so that the browser knows how to interpret the content.

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Page Title</title>
    </head>
    <body>
        Content of the webpage.
    </body>
</html>
  • <!DOCTYPE html>: Specifies an HTML5 document.
  • <html>: Encloses all the HTML code.
  • <head>: Contains metadata, scripts, stylesheets, and the title.
  • <title>: Title of the webpage, visible in the browser tab.
  • <body>: Contains the actual visible content.

2. Textformatierung

Diese Tags helfen dir dabei, Text hervorzuheben und zu strukturieren:

<p>Dies ist ein Absatz.</p>
<b>Fetter Text</b>
<strong>Wichtiger fetter Text</strong>
<i>Kursiver Text</i>
<em>Betonter kursiver Text</em>
<u>Unterstrichener Text</u>
<mark>Markierter Text</mark>
  • <p>: Absatz.
  • <b>: Fett, aber ohne inhaltliche Hervorhebung.
  • <strong>: Wichtig hervorgehobener (fetter) Text mit semantischer Bedeutung.
  • <i>: Kursiv, ohne spezielle semantische Bedeutung.
  • <em>: Betont (kursiv), semantisch wichtig.
  • <u>: Unterstrichen, wird aber selten verwendet, da Unterstreichung oft mit Links assoziiert wird.
  • <mark>: Hervorhebung wie mit einem Textmarker.

2. Text Formatting

These tags help you highlight and structure text:

<p>This is a paragraph.</p>
<b>Bold Text</b>
<strong>Important Bold Text</strong>
<i>Italic Text</i>
<em>Emphasized Italic Text</em>
<u>Underlined Text</u>
<mark>Highlighted Text</mark>
  • <p>: Paragraph.
  • <b>: Bold, but without semantic emphasis.
  • <strong>: Strongly emphasized (bold) text with semantic meaning.
  • <i>: Italic, without special semantic meaning.
  • <em>: Emphasized (italic), semantically important.
  • <u>: Underlined, but rarely used as underlining is often associated with links.
  • <mark>: Highlighted like with a marker.

3. Überschriften

Überschriften strukturieren Inhalte hierarchisch. <h1> ist die wichtigste Überschrift, <h2> bis <h6> folgen in absteigender Wichtigkeit.

<h1>Überschrift 1</h1>
<h2>Überschrift 2</h2>
<h3>Überschrift 3</h3>
  • <h1> bis <h6>: Überschriften (1 = größte, 6 = kleinste).

5. Links und Bilder

Links verbinden deine Seite mit anderen Seiten. Bilder werden eingebunden, um visuelle Inhalte anzuzeigen.

<a href="https://example.com">Ein Link</a>
<img src="bild.jpg" alt="Ein Bild" width="200" height="100">
  • <a>: Link-Tag. href bestimmt das Ziel.
  • <img>: Bild-Tag. src ist die Bildquelle, alt der Alternativtext.

5. Links and Images

Links connect your page to other pages. Images are embedded to display visual content.

<a href="https://example.com">A Link</a>
<img src="image.jpg" alt="An Image" width="200" height="100">
  • <a>: Link tag. href determines the target.
  • <img>: Image tag. src is the image source, alt is the alternative text.

6. Tables

Tables organize data into rows and columns. They are especially suitable for tabular data, not for general layout.

7. Formulare

Formulare sammeln Benutzereingaben und senden diese an einen Server.

<form action="/submit" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<input type="submit" value="Abschicken">
</form>
  • <form>: Enthält das gesamte Formular. action gibt das Ziel an, method die Versandart (GET/POST).
  • <label>: Beschriftet ein Eingabefeld.
  • <input>: Eingabefeld, dessen type bestimmt, welche Art von Eingabe möglich ist (Text, Passwort, Checkbox, usw.).
  • <button>: Erzeugt eine Schaltfläche. Mit type="submit" wird sie zum Absenden des Formulars verwendet.

7. Forms

Forms collect user inputs and send them to a server.

<form action="/submit" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<input type="submit" value="Submit">
</form>
  • <form>: Contains the entire form. action specifies the target, method the submission method (GET/POST).
  • <label>: Labels an input field.
  • <input>: Input field, whose type determines the kind of input possible (text, password, checkbox, etc.).
  • <button>: Creates a button. With type="submit", it is used to submit the form.

8. Multimedia

HTML5 erlaubt das direkte Einbinden von Videos und Audiodateien.

<video width="320" height="240" controls>
<source src="video.mp4" type="video/mp4">
Ihr Browser unterstützt dieses Video nicht.
</video>

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
Ihr Browser unterstützt dieses Audio nicht.
</audio>
  • <video>: Video einbinden, mit controls für Abspielfunktionen.
  • <audio>: Audio einbinden, ebenfalls mit controls.

9. Semantische Tags

Semantische Tags geben dem Inhalt eine Bedeutung und helfen Suchmaschinen und Screenreadern bei der Interpretation.

<header>Kopfbereich (Logo, Hauptmenü)</header>
<nav>Navigationsbereich</nav>
<main>Hauptinhalt</main>
<section>Thematischer Abschnitt</section>
<article>Eigenständiger Artikel</article>
<aside>Zusätzliche Info am Rand</aside>
<footer>Fußzeile (Copyright, Kontakt)</footer>
  • <header>: Kopfbereich einer Seite oder eines Abschnitts.
  • <nav>: Navigationselement, z. B. ein Menü.
  • <main>: Hauptinhalt, der sich deutlich vom Header, Footer und Sidebar unterscheidet.
  • <section>: Thematischer Abschnitt.
  • <article>: Unabhängiger, selbstständiger Inhalt, z. B. ein Blogpost oder eine Zeitungsartikel.
  • <aside>: Informationen am Rand, z. B. Werbung, weiterführende Links oder ein Glossar.
  • <footer>: Fußzeile, meist mit Urheberangaben, rechtlichen Hinweisen oder Kontaktinformationen.

9. Semantic Tags

Semantic tags give the content meaning and help search engines and screen readers interpret it.

<header>Header Area (Logo, Main Menu)</header>
<nav>Navigation Area</nav>
<main>Main Content</main>
<section>Thematic Section</section>
<article>Independent Article</article>
<aside>Additional Info at the Side</aside>
<footer>Footer (Copyright, Contact)</footer>
  • <header>: Header area of a page or section.
  • <nav>: Navigation element, e.g., a menu.
  • <main>: Main content that is distinct from the header, footer, and sidebar.
  • <section>: Thematic section.
  • <article>: Independent, self-contained content, e.g., a blog post or newspaper article.
  • <aside>: Information at the side, e.g., advertisements, additional links, or a glossary.
  • <footer>: Footer, usually with copyright information, legal notices, or contact information.

Glossar

Tag
Ein Baustein in HTML, der von spitzen Klammern umschlossen ist, z.B. <p>.
Attribut
Ein Zusatz zu einem HTML-Tag, der zusätzliche Informationen bereitstellt, z.B. href bei <a>.

Glossary

Tag
An HTML building block enclosed in angle brackets, e.g., <p>.
Attribute
An addition to an HTML tag that provides additional information, e.g., href in <a>.
